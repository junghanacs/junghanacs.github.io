## install wcite {#install-wcite}

[wcite - wikicite.org](http://wikicite.org/wcite/)

```text
sudo npm install -g wcite
```

아래와 같이 활용.

```text
➜ wcite Q55239420
Q55239420: Shelley, M. (1818). Frankenstein (1st ed.). (Original work published 1818)
```
